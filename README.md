# Go Programming Lab (TYBCA)

This repository contains basic Go (Golang) programs for TYBCA practical lab.

## 📌 Topics Covered
- Basic Programs
- Functions
- Recursion
- Arrays
- Slices
- Structures
- File Handling
- Pointers

## 📂 Program Sets

### ✅ Set A
- Largest and Smallest in Array
- Book Details using Structure
- Slice Initialization

### ✅ Set B
- Multidimensional Slice
- Sorting Array
- Student Details with Total and Average

### ✅ Set C
- Matrix Multiplication
- Employee with Maximum Salary
- Slice Operations (append, remove, copy)

## 💻 Language Used
- Go (Golang)

## 🎯 Purpose
This repository is created for practice and academic learning.
